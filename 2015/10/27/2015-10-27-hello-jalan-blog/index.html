3.安装jekyll

在安装jekyll前先确保gem已正确安装

确保gem安装后，开始安装jekyll gem
在命令行执行：


gem install jekyll

gem source -r http://rubygems.org/

gem source -a http://ruby.taobao.org/

jekyll new myblog

cd myblog
