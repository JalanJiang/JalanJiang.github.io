5 迭代器

Iterable - 可迭代对象

可以直接作用于 for 循环的对象统称为可迭代对象。

包括：

  • list
  • tuple
  • dict
  • set
  • str
  • generator

Iterator - 迭代器

特点：

  1. 可以被 next() 函数调用并不断返回下一个值
  2. Iterator 对象表示一个数据流
  3. Iterator 的计算是惰性的，只有在需要返回下一个数据时它才会计算
