1044错误

错误提示如下：

产生错误原因：备份数据库时使用的用户没有 locak table 权限。

mysqldump 在转储表之前会发出锁表命令，由此来保证备份吼的数据一致性。

解决方法1

换一个有 locak table 权限的用户呗。

解决方法2

使用 --skip-lock-tables 跳过锁表操作。
