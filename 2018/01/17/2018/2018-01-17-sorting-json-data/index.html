json_encode 无序问题

Jan 17, 2018 | Web屠龙刀

问题描述

最近在项目中碰到一个问题：将 PHP 数组按某字段排列后使用 json_encode 对其进行序列化，再写入 redis 后发现 json 数据并未按照 PHP 数组的顺序排列。

于是查了下资料才发现 —— json是无序的。

JavaScript 或者说 json 标准中的 Object 是不保证顺序的，json 库任意改变 key 都是符合标准的行为，任何特定实现中对 key 顺序的处理（保留／排序／乱序）都不应当被依赖。
