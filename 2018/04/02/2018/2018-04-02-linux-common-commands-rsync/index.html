rsync 简述

  • 对位于不同位置的文件盒目录进行同步（远端同步／备份）
  • 优势（与 cp 相比）：
    • 差异计算
    • 压缩技术
    • 加密技术

rsync 备份姿势

将源目录复制到目的端

  • -a：归档
  • -v：在 stdout 上打印出细节信息或进度

将数据备份到远程服务器或主机

  • rsync 使用 ssh 连接远程主机
    • user：代表用户名
    • host：代表远程主机的IP地址或主机名
    • PATH：指定需要从中复制数据的远程主机上的路径

压缩数据

-z 指定在网路传输时压缩数据。
